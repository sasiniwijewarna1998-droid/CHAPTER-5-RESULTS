## Thesis Results File Tracking

This repository contains the result files, intermediate outputs, and missing file checklist for Chapter 05 of the MSc thesis:

**Comparative Genomic Analysis and Epitope Prediction of Dengue and Zika Virus Variants in Sri Lanka and South-East Asia for Designing Future Rational Vaccine Candidates**

The file `thesis_results_file_tracking.json` provides a structured section-wise record of:

- available result files
- missing files
- analysis descriptions
- thesis section mapping

This checklist will be updated as additional outputs are generated and uploaded.
