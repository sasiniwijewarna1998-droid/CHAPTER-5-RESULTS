# Validation
Contains plots validating the quality of the refined 3D model, including Ramachandran plot (stereochemical quality), ProSA (overall model quality), and AlphaFold metrics (pLDDT/PAE).

**Required Files:** 
- `ramachandran_plot.png`
- `prosa_z_score_plot.png`
- `pLDDT_plot.png`
- `PAE_plot.png`
